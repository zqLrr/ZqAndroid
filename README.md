# ZqAndroid

作为自己个人技术wiki网站，主要技术栈采用 mkdocs 来生成网站。[mkdocs使用部署文档](https://hellowac.github.io/mkdocs-docs-zh/)

使用步骤：

```bash
#本地测试
mkdocs build
# 远程部署
mkdocs gh-deploy
```
